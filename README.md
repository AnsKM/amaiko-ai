## Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd amaiko-ai
```

2. Create and activate virtual environment:
```bash
python3 -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```
